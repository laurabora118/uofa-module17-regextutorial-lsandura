Regex Explained

The Regex can be somewhat fun, you can make it unique

In speaking language there are sounds that are patterns, regular expresssions are like this but are written patterns.

Regex takes a sequence of characters to define a pattern, uses a regular type of flow but also is generalized

Pretend you want to look up a code of 3 numbers then a dash then a three numbers such as 123-456.

With the regex you could do that with like this \d\d\d-d\d\d\ to find the pattern.
